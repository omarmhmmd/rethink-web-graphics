Rethink: Web

  • About

    The Rethink: Web logo was designed as a piece to reflect not only the ideologies behind Rethink: Web but as a symbol regarding the speculative future of the relationship between the web and the grid. As graphic designers today are attempting to break the grid, so too are the designers of the web. Individual pieces within the grid itself represent the constraint of the four cornered two dimensional screen all browsers adhere by. The organic curves and form generated through user input then create an abstracted grid that does not allow for the basic ontology of a grid, stating there is room to modify the web’s identity and its relationship to the grid and eventually morph into a new design paradigm. Through this web based application, designers of Rethink: Web are encouraged to create a new logo for each new design solution related to the program, allowing for the design system of Rethink: Web to be continuous and ever-changing, just like the web. This application is part of the Rethink: Web series on Open Space.
